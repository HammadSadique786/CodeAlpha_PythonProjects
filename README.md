# CodeAlpha Python Programming Projects

This repository contains multiple Python programming projects completed as part of the CodeAlpha Python Programming Internship.

## Projects Included

### 1. Hangman Game
A simple text-based game where the player guesses a hidden word one letter at a time.

#### Features:
- Random word selection
- 6 wrong attempts limit
- Letter-by-letter guessing
- Win and lose conditions

---

### 2. Stock Portfolio Tracker
A program that calculates total investment value based on user input and predefined stock prices.

#### Features:
- User input for stock name and quantity
- Predefined stock prices
- Total investment calculation
- Saves result in a text file

---

### 3. Basic Chatbot
A simple rule-based chatbot that responds to user inputs.

#### Features:
- Predefined responses
- Basic conversation handling
- Help and exit commands

---

## Technologies Used

- Python
- Loops
- Conditional Statements
- Functions
- Lists & Dictionaries
- File Handling
- Random Module
